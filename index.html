Capstone Course

The CS-499 Computer Science Capstone course at SNHU serves as the final milestone for computer science majors. In this course, I was tasked with refining and enhancing prior coursework to demonstrate mastery across three core categories:

  • Software Engineering and Design – Building maintainable, scalable, and well-structured systems.
  • Algorithms and Data Structures – Designing efficient code for computational problems.

This course required applying industry standards, documenting enhancements, conducting code reviews, and preparing professional presentations to demonstrate my growth as a computer scientist ready for industry challenges.
